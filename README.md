# Starter Project

This repository is provided as a starting point for our training.

The goal of this project is to make sure everyone starts with the same basic structure, so we can focus on learning and building instead of spending time setting up files.

Feel free to modify anything you need:
- HTML
- CSS
- Bootstrap components
- Images
- Folder structure (if needed)

This is **NOT** the final project and **NOT** the required design.

Think of it as a blank canvas that helps you get started faster. You're free to customize it, improve it, and build your own version during the training.
